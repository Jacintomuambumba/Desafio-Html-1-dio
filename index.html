Estrutura Basica do HTML

Voltar

O HTML (Hypertext Markup Language) é a linguagem de marcação fundamental para criar páginas web.

Sua estrutura básica consiste em elementos aninhados que formam a estrutura da página.

Cada elemento é marcado por tags, como "html, head, title, body," etc.

Listas ordenadas e Não ordenadas

Voltar

Em HTML, as listas ordenadas e não ordenadas são criadas usando as tags "ul" (unordered list) e ol (ordered list), respectivamente. Essas tags são combinadas com as tags li (list item) para definir os itens individuais da lista.

Oque São tags

Voltar

As tags são elementos fundamentais em HTML que envolvem e estruturam o conteúdo de uma página web. Elas são usadas para definir elementos e atributos, indicando como o navegador deve interpretar e exibir o conteúdo.

Links

Voltar

Links em HTML são criados com a tag "a" e usam o atributo `href` para definir o destino, como URLs, âncoras ou endereços de e-mail. Essenciais para navegação, podem apontar para páginas externas ou internas, além de permitir interatividade, estilo e semântica na construção de páginas web.
